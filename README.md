# AI Tools for Software Development Productivity (Team 9)

This is a UI prototype demonstrating how AI tools (ChatGPT and GitHub Copilot) 
can support software development tasks such as coding, testing, debugging, 
and documentation.

## Tech Stack

- React (Vite)
- Tailwind CSS
- GitHub collaboration

---

## How to Run

npm install
npm run dev

---

## Project Scope (3 ECTS)

This project demonstrates an AI-assisted software development workflow 
through a frontend UI prototype.

Outputs are simulated. No backend AI API integration is used.

---

## Planned Modules

- Dashboard (progress overview)
- AI Assistant (ChatGPT vs Copilot workflow simulation)
- Test Generator (AI-assisted test suggestions)
- Docs Helper + Guidelines (best practices and documentation support)
